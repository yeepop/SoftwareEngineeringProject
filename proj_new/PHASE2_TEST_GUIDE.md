### 2. 送養表單測試 (RehomeForm.vue)

**測試步驟:**
前往: http://localhost:5173/rehome-form

**測試案例 2.1: 步驟導航**
- [ ] 驗證顯示 4 個步驟指示器

**測試案例 2.2: 步驟 1 - 基本資訊**

- [ ] 選擇物種: 貓/狗 (必填)

- [ ] 填寫顏色 (選填)
- [ ] 填寫描述 (必填)
- [ ] 不填必填欄位點下一步

- [ ] 點下一步
- [ ] 驗證成功進入步驟 3

**測試案例 2.4: 步驟 3 - 醫療資訊**

- [ ] 勾選已結紮 checkbox

**測試案例 2.5: 步驟 4 - 確認送出**

### 4. 整合測試

**測試案例 4.1: 完整送養流程**
1. [ ] 登入系統
2. [ ] 點擊導航列的送養按鈕
3. [ ] 填寫完整送養表單 (4 個步驟)
4. [ ] 上傳 2-3 張照片
5. [ ] 送出審核
6. [ ] 在我的送養列表中查看
7. [ ] 驗證狀態為審核中
8. [ ] 查看詳情頁面
9. [ ] 驗證所有資訊正確顯示

**測試案例 4.2: 草稿恢復流程**

5. [ ] 重新開啟送養表單
6. [ ] 驗證自動載入草稿
7. [ ] 繼續填寫完成
8. [ ] 送出審核
9. [ ] 驗證草稿已清除

**測試案例 4.3: 編輯送養資訊**
